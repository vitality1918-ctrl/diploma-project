# Diploma NER Project

Этот репозиторий содержит проект по распознаванию именованных сущностей (NER) на основе BiLSTM для договорных текстов.

## Структура проекта

- **final_notebook/** — финальный ноутбук, итоговые эксперименты, результаты и выводы. Подробное описание см. в `final_notebook/README.md`.
- **experiments_notebook/** — экспериментальный ноутбук, черновики, тесты, промежуточные эксперименты. Подробнее — в `experiments_notebook/README.md`.
- **contracts_ner_env/** — окружение и вспомогательные скрипты для обработки данных и обучения моделей.
- **audio_realtime_project/**, **contracts_ner_project/** и другие — дополнительные проекты и ресурсы.

## Быстрый старт

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/vitality1918-ctrl/diploma-project.git
   ```
2. Ознакомьтесь с финальным ноутбуком и результатами в папке `final_notebook/`.
3. Для запуска экспериментов используйте инструкции из соответствующих README в подпапках.
