# Diploma NER Project

Этот репозиторий содержит проект по распознаванию именованных сущностей (NER) на основе BiLSTM для договорных текстов.


## Структура проекта

- **final_notebook/** — финальный ноутбук, итоговые эксперименты и результаты. Подробное описание см. в final_notebook/README.md.
- **experiments_notebook/** — ноутбук, содержит промежуточные эксперименты при создании модели. Подробнее — в experiments_notebook/README.md.

## Быстрый старт

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/vitality1918-ctrl/diploma-project.git
   ```
2. Ознакомьтесь с финальным ноутбуком и результатами в папке final_notebook/.
3. Для запуска экспериментов используйте инструкции из соответствующих README в подпапках.
