# Финальный ноутбук NER

## Запуск в Google Colab

1. Откройте ноутбук по ссылке:
   https://colab.research.google.com/github/vitality1918-ctrl/diploma-project/blob/main/final_notebook/final_inference.ipynb

2. Нажмите:
   Runtime → Restart and run all (Среда выполнения → Перезапустить и выполнить всё)

3. При необходимости загрузите файлы модели через интерфейс Colab (или используйте ячейку для загрузки файлов из Google Drive/GitHub).

## Требуемые файлы
- model/bilstm_ner_best.pt
- vocab/char_vocab.json
- vocab/label_vocab.json
- data/sample_contract.txt

Файлы должны быть размещены в соответствующих папках внутри final_notebook. Если вы запускаете ноутбук локально, убедитесь, что структура папок совпадает с репозиторием.
